# BoxLang Playwright E2E Testing

End-to-end UI tests for BoxLang web applications using [Playwright](https://playwright.dev/java/) and [TestBox](https://www.ortussolutions.com/products/testbox).

## Prerequisites

- [BoxLang](https://boxlang.io/) 1.12+
- [CommandBox](https://www.ortussolutions.com/products/commandbox)
- Java 21 (JRE or JDK)
- [Maven](https://maven.apache.org/) 3.x

## Project Structure

```
├── Application.bx              # Root app config (loads lib/ JARs)
├── box.json                     # CommandBox dependencies (TestBox)
├── server.json                  # BoxLang server config (port 4099)
├── tests/
│   ├── pom.xml                  # Maven config for Playwright JARs
│   ├── lib/                     # Playwright JARs (generated by Maven)
│   ├── Application.bx           # Test app config (loads lib/ JARs)
│   ├── runner.bxm               # TestBox HTML runner
│   ├── resources/
│   │   ├── BaseSpec.bx          # Shared test config (baseURL, browser defaults)
│   │   └── PlaywrightService.bx # Playwright wrapper API
│   └── specs/
│       └── E2E/
│           └── HomePageTest.bx  # Home page test suite
```

## Setup

### 1. Install CommandBox Dependencies

```bash
box install
```

This installs TestBox into the `testbox/` directory.

### 2. Install Playwright Maven Dependencies

```bash
cd tests
mvn install
```

This downloads the Playwright JARs and their transitive dependencies into `tests/lib/`. The `Application.bx` files are already configured to load JARs from this directory via `this.javaSettings.loadPaths`.


### 3. Start the Application

Make sure the target web application is running. For local development:

```bash
box server start
```

The server starts on the port defined in `server.json` (default: `4099`). Update `baseURL` in `tests/resources/BaseSpec.bx` if your app runs on a different host/port.

## Running Tests

### Run All Tests (Browser)

Start the BoxLang server and open the TestBox runner in a browser:

```
http://localhost:4099/tests/runner.bxm
```

### Run All Tests (CLI)

```bash
box testbox run
```

### Run a Single Test Suite

```bash
box testbox run runner="http://localhost:4099/tests/runner.bxm" bundles="tests.specs.E2E.HomePageTest"
```

### Run with a Specific Reporter

```
http://localhost:4099/tests/runner.bxm?reporter=json
```

Available reporters: `simple`, `json`, `junit`, `dot`, `min`, `text`, `xml`.

## Browser Configuration

Override browser defaults per test suite at the top of `run()`:

```boxlang
function run() {
    // Run headless with no slow-motion delay
    variables.browserDefaults = { "headless": true, "slowMo": 0 };

    // Run only on Chromium
    variables.browsers = [ "chromium" ];

    forEachBrowser( "My Tests", ( browserName ) => {
        // ...
    } );
}
```

Defaults in `BaseSpec.bx`:

| Setting    | Default     | Description                              |
|------------|-------------|------------------------------------------|
| `headless` | `false`     | Run browser without visible window       |
| `slowMo`   | `0`         | Milliseconds delay between actions       |
| `browsers` | `["chromium", "firefox", "webkit"]` | Engines to test against |

## PlaywrightService API

The `PlaywrightService.bx` wraps Playwright's Java API. All methods return booleans or values — use TestBox's `expect()` for assertions.

### Navigation
- `goTo( url )` — navigate (prepends `baseURL` for relative paths)
- `reload()` — reload current page
- `goBack()` — browser back

### Interactions
- `click( selector )`, `doubleClick( selector )`, `hover( selector )`
- `check( selector )`, `uncheck( selector )`
- `fill( selector, value )`, `type( selector, value )`
- `press( key )` — keyboard key (e.g. `"Enter"`)
- `selectOption( selector, value )`, `dragAndDrop( source, target )`

### Queries
- `isVisible( selector )` → `boolean`
- `hasText( selector, text )` → `boolean`
- `hasURL( url )` → `boolean`
- `getText( selector )` → `string`
- `getAttribute( selector, attribute )` → `string`
- `getTitle()` → `string`
- `getCurrentURL()` → `string`
- `getPage()` — raw Playwright Page object

### Waiting
- `waitForSelector( selector )`
- `waitForURL( url )`

### Lifecycle
- `close()` — release browser and Playwright resources
